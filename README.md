# mini-comunity-f
微信小程序-兴趣社群前后端分离项目

前端技术选型：
微信原生wxxml+wxss+js
组件库vant

目标：为校园学生提供一个轻量、高效的兴趣社群互动平台，支持社群创建与管理、成员招募、话题讨论、线下活动组织与报名、公告推送、成员等级体系、活动相册分享等功能。
核心功能：
1.	兴趣社群创建、编辑、解散、社群管理员管理（社群主/管理员角色）。
2.	成员加入/退出、成员搜索与招募（公开/审核加入）。
3.	话题发布、评论、点赞、置顶、删除、回帖分页。
4.	线下活动发布、报名、人数上限、签到（可选二维码签到或签到码）。
5.	社群公告推送（群内公告与订阅消息/模板消息推送）。
6.	成员等级体系（积分、徽章、成长值，触发规则）。
7.	活动相册（活动后照片上传、查看、点赞、评论）。
8.	基本用户认证（学号/学校邮箱/微信用户信息映射）与个人资料页。
9.	后台/管理员管理（用于论文演示可做简易后台页面或云函数管理），web管理页面。
